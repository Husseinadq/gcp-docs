## What this section covers

This section explains the concepts that confuse new users most:

- what a project really is
- how billing is attached
- how regions and services fit together
- which services are the best first services to learn

## Good default mindset

A simple first GCP stack for many teams is:

- Cloud Run for application runtime
- Cloud Storage for files and assets
- Cloud SQL for relational data
- Secret Manager for secrets
- Cloud Logging and Monitoring for visibility

That stack is not correct for every team, but it is a strong starting point because it keeps operational complexity low.
